# 🚀 Збірка та Публікація Planer

Оскільки наш додаток написаний на **React + Electron** і використовує **Firebase**, ти можеш обрати два шляхи: зробити з нього десктопну `.exe` програму або розмістити як звичайний сайт в інтернеті.

---

## 1. Як зібрати `.exe` файл (Десктопна версія)

У проекті вже налаштований `electron-builder`, тому збірка дуже проста.

1. Відкрий термінал у папці проекту (`c:\Users\boris\Desktop\planer`).
2. Запусти команду:
   ```bash
   npm run build
   ```
3. Почнеться збірка:
   - Спочатку Vite збере оптимізовану версію React-коду (у папку `dist`).
   - Потім `electron-builder` спакує все у готовий інсталятор.
4. **Готово!** Твій `.exe` файл з'явиться у папці `dist-electron/`. 
   - Файл буде називатися приблизно `Planer Setup 1.0.0.exe`.
   - Можеш відправляти його друзям, встановлювати та користуватись.

> *Примітка:* Якщо захочеш змінити іконку програми, поклади файл `icon.ico` (256x256) у папку `build/` (створи її в корені проекту) та запусти збірку знову.

---

## 2. Як "захостити" як веб-сайт (Web версія)

Оскільки ми зробили безпечні перевірки на `electronAPI` (`try/catch` з fallback'ом на localStorage та Firebase), додаток **може працювати прямо в браузері**!

Найпростіший спосіб безкоштовно захостити Planer — це **Vercel** або **Firebase Hosting**.

### Варіант А: Хостинг через Vercel (Найпростіший)

1. Зареєструйся на [vercel.com](https://vercel.com/) (можна через GitHub).
2. На комп'ютері встанови Vercel CLI:
   ```bash
   npm install -g vercel
   ```
3. У терміналi в папці проекту напиши:
   ```bash
   vercel
   ```
4. Увійди в акаунт, обери стандартні налаштування (просто натискай Enter).
5. Після цього напиши команду для продакшн-публікації:
   ```bash
   vercel --prod
   ```
6. Vercel видасть тобі готове посилання (URL), за яким твій Planer буде доступний усьому світу!

### Варіант Б: Firebase Hosting

Оскільки ми вже використовуємо Firebase для бази даних, логічно тримати все в одному місці.

1. Встанови Firebase CLI:
   ```bash
   npm install -g firebase-tools
   ```
2. Авторизуйся у Firebase:
   ```bash
   firebase login
   ```
3. Ініціалізуй хостинг у проекті:
   ```bash
   firebase init hosting
   ```
   - Обери свій проект (напр. `planer-app`).
   - На питання `What do you want to use as your public directory?` напиши: **`dist`** (це важливо!).
   - На питання `Configure as a single-page app?` напиши: **`y`** (Yes).
   - На питання про GitHub Actions можеш відповісти `N` (No).
4. Збери проект локально:
   ```bash
   npm run build
   ```
   *(vite створить папку `dist` з готовим веб-додатком)*
5. Завантаж на Firebase:
   ```bash
   firebase deploy --only hosting
   ```
6. Firebase видасть тобі URL твого сайту (наприклад `https://planer-app-XXXXX.web.app`).

---

## Підсумок

- Для **Windows-додатка (`.exe`)**: просто напиши `npm run build` і забери файл з папки `dist-electron`.
- Для **Сайту**: збери проект і залий папку `dist` на Vercel або Firebase Hosting. Усі твої записи будуть синхронізуватись між сайтом і програмою на комп'ютері завдяки базі даних! 🔥
